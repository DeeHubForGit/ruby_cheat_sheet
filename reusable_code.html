Gems

  • Reusable libraries that add features to Ruby
  • Used to avoid writing everything from scratch
  • Installed using the gem command
  • Commonly used in Ruby and Rails projects
gem install rails

Gemfile & Bundler

  • Projects list gems in a Gemfile
  • bundle install installs all project dependencies
  • Preferred over running gem install manually for projects
bundle install

Common Gems

Common Ruby gems:

  • Rails: Web application framework.
  • RSpec: Testing Ruby code.
  • Devise: User authentication.
  • Puma: A web server for Ruby and Rails applications.

Common Rails gems:

  • Active Record: Database interaction.
  • Action Mailer: Sending emails.
  • Active Job: Background job processing.
  • Active Storage: File uploads.

Require / require_relative

  • require loads installed libraries (standard library or gems)
  • require_relative loads a local file relative to the current file's location
  • You usually put require lines at the top of a file
require "json"            # installed library
require_relative "helpers"      # local file

data = JSON.parse('{"name":"Joe"}')
puts data["name"]

Rails

  • Web application framework written in Ruby
  • Follows MVC (Model View Controller)
  • Distributed as a gem using gem install rails
